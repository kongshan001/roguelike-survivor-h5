### Tier 2: 新基础武器 (v2.1)

**设计规格完成度: 0% -- 尚未设计**

**竞品调研对标:**
- P1 第 5 项: "新基础武器 x2 (回旋镖/毒雾)"
- 回旋镖已在 v1.5.0 实现 (作为第 7 种基础武器)
- 剩余候选: 毒雾(PoisonMist) / 光束(LaserBeam) / 召唤(Summon)

**推荐: 毒雾 (PoisonMist)**
- 攻击模式: 持续伤害区域 -- 以玩家为中心释放毒雾，范围内敌人持续受到伤害
- 定位: 控场 + DOT 组合型
- 与现有武器差异: 圣经是瞬间伤害, 冰冻光环是减速, 毒雾是纯 DOT (无减速/冰冻)
- 进化候选: 毒雾 + 圣经 = 腐蚀领域 (大范围 DOT), 毒雾 + 磁铁 = 瘟疫磁力 (毒雾范围吸引宝石)
- Synergy 候选: 毒雾 + 护甲 = 猛毒护甲 (毒雾范围内受伤减少), 毒雾 + 再生 = 腐蚀再生 (毒雾中再生加速)

**前端实现工作量: 约 3-4 天**

## 4. 决策记录

- v2.0 核心定为幸运硬币 + 无尽模式，因为这两项的设计规格已 100% 完善，前端可立即实现，无需额外设计等待
- 新武器推荐毒雾而非光束/召唤，原因: (1) 毒雾是 DOT 区域型，与现有 7 种武器完全不同; (2) 进化候选与圣经/磁铁形成有趣组合; (3) 实现复杂度适中 (区域判定 + DOT 逻辑可复用火焰法杖的点燃机制)
- 新敌人选择护盾型 + 自爆型，原因: (1) 护盾型创造"绕背攻击"策略，与环绕武器 (圣水/圣经) 天然协同; (2) 自爆型创造"距离管理"策略，与冲刺闪避系统联动; (3) 两者都不需要新系统支持 (护盾 = 正面伤害减免, 自爆 = 死亡时范围伤害)
- 多关卡系统推迟到 v2.2 而非 v2.0，原因: (1) 工作量最大 (8-12 天); (2) 依赖新敌人 + 新 Boss 的设计; (3) 幸运硬币 + 无尽模式已经是 v2.0 足够的内容量
- Weapon Mastery 推迟到 v2.3 而非 v2.0，原因: (1) 是长期留存系统，不急迫; (2) 需要先有足够的武器数量才有效果 (当前 7 种足够但 8 种更好); (3) 需要商店面板 UI 扩展
- v2.0 不包括新角色，原因: 当前 3 角色已覆盖均衡/坦克/速攻三角，新角色需要新机制 (如专属技能) 才有差异化，属于 v2.1+ 范畴
