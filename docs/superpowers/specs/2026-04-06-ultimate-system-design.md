# 终极技能(Ultimate)系统设计规格

**版本**: v2.5 设计规格
**日期**: 2026-04-06
**Drive**: #30
**来源**: 竞品调研 Soulstone Survivors 的 Active Skill 系统 + 设计积压新方向

---

## 1. 设计概述

### 1.1 问题陈述

当前游戏操控维度仅有"移动"（虚拟摇杆/WASD）和"冲刺闪避"（Space/DASH按钮）。所有武器自动攻击，玩家在 5 分钟内的操作深度有限：移动走位 + 时机选择升级。竞品 Soulstone Survivors 证明"手动释放大招"能显著提升操控参与感和"高光时刻"的爽感，且在 5 分钟短局中恰好创造 1-2 个戏剧性时刻。

### 1.2 核心设计

终极技能(Ultimate)是每个角色独有的充能型大招。玩家通过击杀敌人和承受伤害积累充能，满充能后按 Q 键（PC）或点击 ULT 按钮（移动端）释放。释放后角色进入 3-5 秒的强化状态，产生大范围视觉冲击和大量伤害。充能每局从零开始，不跨局保留。

### 1.3 设计约束

- 充能机制简单直觉：击杀=充能，无需额外 UI 解释
- 自动瞄准：不需要手动瞄准，按 Q 即释放，与移动端兼容
- 单按钮操作：Q 键（PC）/ ULT 按钮（移动端），与 DASH 按钮并列
- 充能归零：每局开始清零，不跨局保留
- 不占升级槽：Ultimate 不是升级选项，是角色固有技能
- 受 weaponDmgMul 影响：与商店武器精通联动

### 1.4 为什么选择 Ultimate 系统作为下一个设计方向

| 标准 | 分析 |
|------|------|
| 操控深度 | 当前只有移动+冲刺，Ultimate 是最直接的操控深度提升 |
| 竞品差异化 | 项目差异化方向之一是"5分钟极致浓缩"，Ultimate 创造浓缩的高光时刻 |
| 移动端兼容 | 自动瞄准+单按钮释放，与 DASH 系统并列，移动端零额外负担 |
| 角色差异化 | 当前3角色仅在属性上有差异，Ultimate 让每个角色有独特的战术定位 |
| 实现成本 | 中等 -- 新增 charge 追踪 + 4种效果 + 1个UI按钮，约200行 |
| 与现有系统协同 | weaponDmgMul(商店) + combo(连击) + screenShake(震动) + synergy(协同) |

## 2. 充能机制

### 2.1 充能来源

| 来源 | 充能值 | 说明 |
|------|--------|------|
| 击杀普通敌人 | +1.5% | 主要来源 |
| 击杀精英敌人 | +5.0% | 鼓励击杀高价值目标 |
| 击杀 Boss | +12.0% | Boss 击杀奖励大量充能 |
| 受到伤害 | +1.0% / 次 | "受挫补偿" -- 被打也有收获 |

### 2.2 标准局充能追踪（预估）

| 时段 | 击杀/分 | 充能/分 | 累积充能 |
|------|---------|---------|---------|
| 0:00-1:00 | ~20 | 30% | 30% |
| 1:00-2:00 | ~30 | 45% | 75% |
| 2:00-2:20 | ~15 | 22.5% | 97.5% |
| 2:20-2:25 | ~5 | 7.5% | 105% -- **首次释放 (~2:25)** |
| 2:25-3:00 | ~30 + 1精英 | 50% | 55% |
| 3:00-4:00 | ~45 + 2精英 | 77.5% | 132.5% -- **二次释放 (~3:50)** |
| 4:00-4:30 | ~25 | 37.5% | 70% |

**结论**: 标准 5 分钟局约 2 次释放。首次 ~2:25（骷髅/幽灵期开始），二次 ~3:50（精英期高潮）。节奏恰好匹配游戏强度曲线。

### 2.3 充能机制详细规格

| 属性 | 数值 | 说明 |
|------|------|------|
| 最大充能 | 100% | 充能条满时可以释放 |
| 充能溢出 | 不溢出 | 超过 100% 的充能被丢弃（不累积到下次） |
| 释放后充能 | 归零 | 每次释放后充能回到 0% |
| 每局初始 | 0% | 不跨局保留 |
| 释放时暂停 | 不暂停 | Ultimate 释放不暂停游戏（与升级面板不同） |

## 3. 角色终极技能设计

### 3.1 魔法师 -- 奥术轰炸 (Arcane Barrage)

**定位**: 大范围清怪型 Ultimate。最适合敌人密集时释放，瞬间清场。

| 属性 | 数值 | 说明 |
|------|------|------|
| 名称 | 奥术轰炸 | -- |
| 图标 | &#10024; (sparkles) | -- |
| 持续时间 | 4 秒 | -- |
| 导弹波次间隔 | 0.4 秒 | 10 波导弹 |
| 每波导弹数 | 4 枚 | 随机选择可见范围内的敌人 |
| 单枚伤害 | 10 | 受 weaponDmgMul 影响 |
| 溅射范围 | 25px | 命中点 25px 内敌人受 50% 伤害 |
| 目标选择 | 自动（随机可见敌人） | 无需瞄准 |
| 导弹速度 | 600px/s | 从画面顶部飞向目标 |
| 总导弹数 | 40 枚 (10波 x 4) | -- |
| 总潜在伤害 | ~400 (直击) + 溅射 | 实际受敌人密度和分布影响 |
| 视觉 | 紫色(#e040fb)能量球从天而降，命中爆炸 | 全屏紫色闪光 |

**充能后DPS提升**: 持续4秒释放40枚导弹，每秒10枚 x 10伤害 = 100 DPS/秒，是基础武器DPS(~20)的5倍。4秒总计约400伤害，足以清空标准密度下的大部分普通敌人。

### 3.2 战士 -- 狂暴战意 (Berserker Rage)

**定位**: 自身强化型 Ultimate。最适合 Boss 战释放，伤害翻倍+无敌是最强爆发窗口。

| 属性 | 数值 | 说明 |
|------|------|------|
| 名称 | 狂暴战意 | -- |
| 图标 | &#128128; (skull) | -- |
| 持续时间 | 5 秒 | 所有强化同时生效 |
| 伤害倍率 | x2.0 | 所有武器伤害翻倍 |
| 攻速倍率 | x1.5 | 所有武器冷却 /1.5 |
| 伤害免疫 | 是 | 完全免疫伤害 |
| 移速加成 | +30% | speed *= 1.3 |
| 视觉 | 红色(#ff1744)脉动光环，玩家放大20%，火焰粒子拖尾 | 全屏红色边缘光 |

**充能后DPS提升**: 基础DPS ~20 -> x2伤害 x1.5攻速 = 60 DPS，是正常的3倍。持续5秒总计约300额外伤害（加上不受伤节省的回复时间）。

### 3.3 游侠 -- 箭雨风暴 (Arrow Storm)

**定位**: 360度防御型 Ultimate。最适合被包围时释放，向四面八方射箭突围。

| 属性 | 数值 | 说明 |
|------|------|------|
| 名称 | 箭雨风暴 | -- |
| 图标 | &#127993; (bow and arrow) | -- |
| 持续时间 | 3.5 秒 | -- |
| 箭矢频率 | 12支/秒 | 总计 42 支 |
| 单支伤害 | 5 | 受 weaponDmgMul 影响 |
| 穿透 | 2 次 | 每支箭命中2个敌人 |
| 扩散方式 | 环形向外扩散 | 从玩家位置向外 360 度 |
| 扩散速度 | 120px/s | -- |
| 最大半径 | 200px | 到达后消失 |
| 总箭矢数 | 42 支 | -- |
| 总潜在伤害 | ~210 (直击) + 穿透 | 穿透后约 420 |
| 视觉 | 绿色(#69f0ae)箭矢轨迹，环形扩散 | 全屏绿色箭雨 |

**充能后DPS提升**: 42支箭 x 5伤害 / 3.5秒 = 60 DPS，与狂暴战意持平但偏向360度均匀分布。

### 3.4 暗影法师 -- 暗影裂隙 (Shadow Rift)

**定位**: 控场聚拢型 Ultimate。最适合配合 AOE 武器（暴风雪/圣光领域）释放，将敌人吸到一起后用范围武器收割。

| 属性 | 数值 | 说明 |
|------|------|------|
| 名称 | 暗影裂隙 | -- |
| 图标 | &#127769; (new moon) | -- |
| 持续时间 | 4 秒 | -- |
| 吸引范围 | 250px | 范围内敌人被拉向中心 |
| 吸引速度 | 60px/s | 向裂隙中心移动 |
| 范围伤害 | 6 DPS | 裂隙内持续伤害，受 weaponDmgMul 影响 |
| 减速 | 吸引范围内敌人减速 40% | 防止敌人逃出 |
| 总潜在伤害 | 24/敌人 x N 敌人 | 聚拢后配合 AOE 收益极高 |
| 视觉 | 深紫色(#7c4dff)漩涡，暗影触手拉拽 | 全屏暗紫色脉冲 |

**设计约束**: 暗影法师是 Secrets 系统中的隐藏角色（需完成 all_evolutions 成就解锁），此 Ultimate 仅在解锁暗影法师后可用。

## 8. 与现有系统交互

### 8.1 商店系统

- weaponDmgMul 提升 Ultimate 伤害（所有伤害型 Ultimate 均受影响）
- shop 中的 HP/speed 等防御升级不直接影响 Ultimate
- gold 升级不影响 Ultimate（Ultimate 不产生金币）

### 8.2 连击系统

- Ultimate 击杀的敌人计入连击计数
- 奥术轰炸一次清场可能直接创造 20-30 连击
- 连击经验加成正常应用

### 8.3 协同系统

- 狂暴战意的伤害倍率与武器协同（如致命飞刀的暴击）乘法叠加
- 暗影裂隙的减速与冰冻光环减速不叠加（取较大值）
- Ultimate 期间武器协同效果正常生效

### 8.4 难度系统

- Ultimate 充能速率不受难度影响（保持一致性）
- Ultimate 伤害受难度影响（困难模式敌人HP更高，但 Ultimate 伤害不变）
- 充能"受伤害"奖励在困难模式下更频繁（受伤更多 -> 充能更快 -> 补偿）

### 8.5 无尽模式

- 充能不跨局保留，每局重新充能
- 无尽模式中更密集的敌人 -> 更快的充能 -> 可能 3-4 次释放
- Boss 周期击杀提供大量充能 (12%)

## 11. 设计决策记录

1. **充能制而非冷却制**: 充能与击杀挂钩创造"越战越强"的正反馈，冷却制在5分钟短局中不够灵活。充能让玩家在密集战斗时更快获得Ultimate，节奏更自然。

2. **自动瞄准而非手动瞄准**: 移动端虚拟摇杆已占用左手拇指，右手拇指用于DASH/ULT按钮，没有额外的操控空间用于瞄准。自动瞄准确保移动端体验与PC一致。

3. **每角色一个Ultimate而非可选**: 简化系统复杂度，强化角色身份认知。玩家选择角色就选择了Ultimate策略，增加角色选择的决策深度。

4. **充能不跨局保留**: 5分钟短局应自包含。跨局保留会让玩家在第一局"为第二局充能"，破坏独立游戏体验。

5. **受伤+1%充能**: "受挫补偿"设计。被打不是纯负面体验，每次受伤都在积攒Ultimate充能。这让困难局面有了一线希望。

6. **Q键而非Shift键**: Shift在某些浏览器中有特殊行为（文本选择），Q键在WASD布局中右手小指可达，且不与任何现有功能冲突。

7. **战士Ultimate不增加新投射物**: 战士是"高HP坦克"定位，其Ultimate强化现有武器而非创造新攻击。这保持了战士"靠硬实力"的角色特征。

8. **暗影裂隙的"吸引"而非"推开**: 吸引敌人聚拢与AOE武器（暴风雪/圣光领域）形成强力协同。推开敌人会让近战武器失效，与游戏核心的"靠近击杀获取宝石"机制矛盾。

9. **Ultimate不计入升级池**: Ultimate是角色的固有技能，不需要通过升级获取或强化。这避免了"必须先拿Ultimate升级"的强制感，让升级选择保持自由。

10. **2次/局的激活频率**: ~2:25首次、~3:50二次。首次在游戏从"轻松"转向"紧张"的拐点，二次在Boss出场前的最后高潮。两次激活恰好创造"开局的期待 -> 中期的释放 -> 高潮的期待 -> Boss前的释放"的节奏弧线。

11. **奥术轰炸使用"随机目标"而非"最近目标**: 随机目标让导弹覆盖更广的区域，创造"天降正义"的视觉冲击。如果只打最近目标，导弹会集中在一个区域，视觉效果不佳。

12. **不设计Ultimate+武器协同**: v2.5首版简化，Ultimate是独立系统。未来可设计"Ultimate期间特定武器伤害额外+x%"的协同，但首版先验证基础体验。

13. **Ultimate击杀计入连击但不额外掉落宝石**: 保持经济平衡。如果Ultimate击杀额外掉落宝石，会破坏金币/经验的精细平衡。连击计入是纯视觉/经验收益，不破坏经济。

14. **screenFlash使用角色专属颜色**: 与SCREEN_SHAKE配合，释放Ultimate时全屏闪光0.3秒 + 震动(intensity:8)。这创造了与Boss出场类似的戏剧性时刻。
