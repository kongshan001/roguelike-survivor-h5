# 设计规格书：第7种基础武器 -- 回旋镖 (Boomerang)

> 版本: v1.5.0+ | 策划 Agent | Drive #18
> 状态: 设计完成，待前端实现

---

## 1. 设计概述

当前6种基础武器覆盖了环绕(圣水)、投射(飞刀)、随机(闪电)、范围(圣经)、锥形(火焰法杖)、光环(冰冻光环)六种攻击模式。缺少**回旋追踪**型武器 -- 投掷后自动追踪最近敌人，命中后飞回玩家手中的回旋镖。

回旋镖的核心差异化特征：
- **双程伤害**：飞出时命中敌人一次，飞回时再命中一次（唯一的"去回双段"武器）
- **自动追踪**：不依赖玩家面朝方向，飞出后自动追踪最近敌人（与飞刀的"面朝方向投掷"形成对比）
- **弧线飞行**：飞出时走弧线，飞回时走直线（视觉辨识度高）
- **单体高爆发**：单次回旋可造成两次命中，是对单体敌人的最高爆发效率

### 为什么选回旋镖而非其他候选

| 候选武器 | 攻击模式 | 不选理由 |
|---------|---------|---------|
| 回旋镖 | 回旋追踪 | **选中** -- 填补"自动追踪+双程伤害"空白 |
| 毒雾 | 持续范围 | 与火焰法杖的锥形持续伤害过于相似 |
| 激光 | 直线穿透 | 与飞刀的穿透过于相似 |
| 召唤物 | 召唤AI | 实现复杂度高，需独立AI逻辑 |
| 弹射球 | 弹射 | 与圣水的环绕碰撞机制重叠 |

## 2. 基础武器：回旋镖 (Boomerang)

### 2.1 攻击模式

| 阶段 | 行为 | 说明 |
|------|------|------|
| **飞出** | 从玩家位置向最近敌人方向投掷，飞行途中自动微调追踪角度（最大偏转45度） | 追踪型飞行 |
| **命中/达到最大距离** | 到达追踪目标或飞行最大距离后，进入返回阶段 | 转折点 |
| **飞回** | 直线飞回玩家当前位置 | 直线返回 |
| **回收** | 到达玩家位置后消失，冷却计时开始 | 回合结束 |

### 2.2 数值表

| 属性 | Lv1 | Lv2 | Lv3 |
|------|-----|-----|-----|
| 投掷数量 | 1 | 2 | 3 |
| 飞出速度 | 280 px/s | 280 px/s | 320 px/s |
| 飞回速度 | 320 px/s | 320 px/s | 360 px/s |
| 命中伤害 | 3 | 4 | 5 |
| 最大飞行距离 | 250 px | 300 px | 350 px |
| 投掷间隔(CD) | 1.8s | 1.4s | 1.0s |
| 穿透 | 0 | 1 | 2 |
| 追踪偏转角 | 30度 | 45度 | 60度 |
| 弧线曲率 | 0.3 | 0.3 | 0.2（更直，更快到达目标） |

### 2.4 精灵视觉

| 元素 | 规格 |
|------|------|
| 形状 | V字形回旋镖（12x6px） |
| 颜色 | 金色(#ffc107) + 棕色木纹(#795548) |
| 飞出特效 | V字旋转 + 橙色尾迹 |
| 飞回特效 | V字反向旋转 + 白色尾迹 |
| 命中特效 | 黄色星星粒子 |

### 2.5 与其他基础武器的对比

| 维度 | 飞刀(Knife) | 回旋镖(Boomerang) |
|------|------------|------------------|
| 目标选择 | 面朝方向 | 自动追踪最近敌人 |
| 飞行路径 | 直线不回头 | 弧线去+直线回 |
| 命中次数 | 单程1次 | 双程2次 |
| 基础伤害(Lv3) | 3/次 | 5/次(双程=10总) |
| 投掷频率(Lv3) | 0.6s | 1.0s |
| 穿透(Lv3) | 1 | 2 |
| DPS(Lv3) | ~15 | ~20(双程) |
| 定位 | 高频低伤 | 低频高伤 |

**回旋镖是DPS最高的单体基础武器**，但投掷间隔最长(1.0s vs 飞刀0.6s)。这意味着它在面对密集小怪时不如飞刀高效（低频大范围），但在面对Boss/精英敌人时更强（单次高爆发）。

## 3. 进化路线

新增2条进化路线，让回旋镖与现有武器配对，保持每种武器恰好参与2条进化路线的对称性。

### 3.1 进化路线A：回旋镖 + 闪电 -> 雷霆回旋 (Thunderang)

**设计理念**：回旋镖的追踪+双程 + 闪电的随机电击 = 每次命中有概率连锁电击周围敌人的追踪型武器。

| 属性 | 数值 |
|------|------|
| 投掷数量 | 4 |
| 命中伤害 | 7 |
| 穿透 | 3 |
| 投掷间隔 | 0.8s |
| 追踪偏转角 | 75度（几乎全向追踪） |
| **闪电链** | 每次命中40%概率对120px范围内随机2个敌人释放电击，伤害8，链式递减50% |
| **双程闪电** | 飞出命中和飞回命中各触发一次闪电链判定 |

```js
// CFG.EVOLUTIONS 新增
{ a: 'boomerang', b: 'lightning', result: 'thunderang', name: '雷霆回旋', icon: '🪃⚡', desc: '追踪+闪电链' }

// CFG.WEAPONS 新增
thunderang: { name: '雷霆回旋', icon: '🪃⚡', desc: '追踪+闪电链', evolved: true }
```

**视觉**：金色V字 + 蓝色电弧缠绕 + 命中时释放黄色闪电

### 3.2 进化路线B：回旋镖 + 火焰法杖 -> 烈焰回旋 (Blazerang)

**设计理念**：回旋镖的弧线飞行 + 火焰法杖的点燃 = 飞行路径留下火焰轨迹，命中敌人点燃，飞回时沿途继续灼烧。

| 属性 | 数值 |
|------|------|
| 投掷数量 | 3 |
| 命中伤害 | 6 |
| 穿透 | 3 |
| 投掷间隔 | 0.9s |
| **火焰轨迹** | 飞行路径每20px留下一个火焰点（持续1.5秒），接触火焰的敌人受2/s持续伤害 |
| **点燃** | 命中敌人点燃2.5秒（3/s持续伤害） |
| **最大火焰点** | 同屏最多20个火焰点 |

```js
// CFG.EVOLUTIONS 新增
{ a: 'boomerang', b: 'firestaff', result: 'blazerang', name: '烈焰回旋', icon: '🪃🔥', desc: '追踪+火焰轨迹' }

// CFG.WEAPONS 新增
blazerang: { name: '烈焰回旋', icon: '🪃🔥', desc: '追踪+火焰轨迹', evolved: true }
```

**视觉**：橙红色V字 + 飞行路径留下火焰粒子 + 命中时火焰爆发

### 3.3 进化系统完整覆盖（8条路线）

更新后每种基础武器恰好参与2条进化路线，形成完美对称网络：

| 武器 | 进化路线A | 进化路线B |

## 4. 与现有系统的交互

### 4.1 协同系统 (Synergy)

回旋镖加入后，为其设计2个武器+被动协同效果：

#### boomerang_magnet: 磁力回旋

| 属性 | 数值 |
|------|------|
| 名称 | 磁力回旋 |
| 图标 | 🪃🧲 |
| 条件 | 回旋镖(boomerang) + 磁铁(magnet) |
| 描述 | 回旋镖飞行路径上自动吸附附近宝石 |
| 定位 | 经济增强+品质生活 |

**设计理由**：
- 回旋镖的飞行距离最远(250-350px)，飞行路径覆盖大量区域
- "飞行路径吸附宝石"让回旋镖不仅是武器，还成为拾取工具
- 与磁铁的"拾取范围+30%"互补而非重叠 -- 磁铁扩大固定范围，磁力回旋在飞行路径上创造移动拾取带
- 实现要点：回旋镖每帧检测飞行路径30px范围内的宝石，将宝石吸引到飞行路径上（而非直接收集，宝石仍需飞到玩家位置才计数）

```js
boomerang_magnet: {
  name: '磁力回旋', icon: '🪃🧲',
  req: { weapon: 'boomerang', passive: 'magnet' },
  desc: '回旋镖飞行路径吸引宝石',
  weaponBonus: { flightPullRange: 30 }
}
```

#### boomerang_crit: 致命回旋

| 属性 | 数值 |
|------|------|
| 名称 | 致命回旋 |
| 图标 | 🪃💍 |
| 条件 | 回旋镖(boomerang) + 暴击戒指(crit) |
| 描述 | 回旋镖可暴击，暴击时回旋镖变大20%并穿透+1 |
| 定位 | 进攻增强 |

**设计理由**：
- 回旋镖是低频高伤武器，暴击对它的收益最高（单次暴击伤害=5x2=10）
- "暴击时变大+穿透+1"让暴击不仅是数值翻倍，还有视觉反馈和功能增强
- 与 knife_crit(飞刀可暴击) 类似但不同 -- 飞刀是高频低伤，回旋镖是低频高伤，暴击的效果体感完全不同
- 实现要点：暴击判定在命中时，飞出命中和飞回命中各判定一次

```js
boomerang_crit: {
  name: '致命回旋', icon: '🪃💍',
  req: { weapon: 'boomerang', passive: 'crit' },
  desc: '回旋镖可暴击',
  weaponBonus: { canCrit: true, critSizeBonus: 1.2, critPierceBonus: 1 }
}
```

## 5. 数值平衡分析

### 5.1 DPS 对比（Lv3，无被动/协同）

| 武器 | 对单体DPS | 对群体DPS(假设5目标) | 说明 |
|------|----------|--------------------|------|
| 飞刀 | ~15 (3dmg/0.6s) | ~15(无AOE) | 高频低伤 |
| 圣水 | ~10 (3球x3dmg/1s) | ~10 | 环绕持续 |
| 闪电 | ~8 (6dmg/0.8s) | ~16(链式2链) | 随机+链式 |
| 圣经 | ~12 (4dmg/1s连续) | ~20(范围多目标) | 范围持续 |
| 火焰法杖 | ~10 (3dmg直+3DOT) | ~15(锥形多目标) | 直伤+DOT |
| 冰冻光环 | ~4 (2dmg/s) | ~4 | 控场为主 |
| **回旋镖** | **~20** (5dmg x 2程/1.0s) | **~10** (穿透2目标) | **低频高伤，单体之王** |

**结论**：回旋镖对单体DPS为20，是所有基础武器中最高的。这是合理的，因为：
1. 投掷间隔最长(1.0s vs 飞刀0.6s)，在miss时浪费的DPS最多
2. 对群体效率低（穿透2 vs 飞刀穿透1，但飞刀投掷频率更高）
3. 追踪机制减少了miss率，但也意味着回旋镖不会"主动选择最优目标"

### 5.2 进化武器 DPS 对比

| 进化武器 | 对单体DPS | 定位 |
|---------|----------|------|
| 雷暴圣水 | ~25 | 持续+链式 |
| 火焰飞刀 | ~30 | 高频穿透+DOT |
| 圣光领域 | ~20 + 脉冲12/4s | 范围+脉冲 |
| 暴风雪 | ~15 + 闪电链8/2s | 控场+闪电 |
| 冰霜飞刀 | ~22 | 减速穿透 |
| 烈焰经文 | ~25 + 脉冲8/3s | 范围灼烧 |
| **雷霆回旋** | **~35** (7x2/0.8s + 闪电链) | **单体爆发+连锁** |
| **烈焰回旋** | **~30** (6x2/0.9s + 轨迹伤害) | **穿透+区域控制** |

**结论**：雷霆回旋(35)是所有进化武器中对单体DPS最高的，这与回旋镖的"单体之王"定位一致。烈焰回旋(30)通过火焰轨迹获得了群体伤害能力。

### 5.3 升级池经济影响

7种基础武器 + 6-7被动 = 13-14种选项，每次升级3选1。

| 场景 | 概率 | 影响 |
|------|------|------|

**Boomerang 类核心逻辑**：

```js
export class Boomerang extends Weapon {
  constructor(owner) {
    super('boomerang', owner);
    this.cd = 1.8; // 初始CD
    this._projectiles = []; // 活跃的回旋镖投射物
  }

  get count() { return this.level; }
  get dmg() { return this.level >= 3 ? 5 : this.level >= 2 ? 4 : 3; }
  get pierce() { return this.level >= 3 ? 2 : this.level >= 2 ? 1 : 0; }

  update(dt, enemies) {
    if (this._timer > 0) { this._timer -= dt; return; }
    // 找最近敌人
    let target = findNearest(this.owner, enemies);
    if (!target) return;

    // 投掷
    const cfg = CFG.BOOMERANG.base.levels[this.level];
    for (let i = 0; i < cfg.count; i++) {
      // 多个回旋镖：目标为最近/次近/第3近的敌人，或相同目标
      this._projectiles.push({
        x: this.owner.x, y: this.owner.y,
        targetX: target.x, targetY: target.y,
        target: target, // 追踪引用
        phase: 'out', // 'out'=飞出, 'return'=飞回
        angle: Math.atan2(target.y - this.owner.y, target.x - this.owner.x),
        speed: cfg.speed,
        returnSpeed: cfg.returnSpeed,
        dmg: cfg.dmg,
        pierce: cfg.pierce,
        maxDist: cfg.maxDist,
        traveled: 0,
        trackAngle: cfg.trackAngle,
        curvature: cfg.curvature,
        hitSet: new Set(), // 已命中的敌人ID（飞出阶段）
        returnHitSet: new Set(), // 已命中的敌人ID（飞回阶段）
      });
    }
    this._timer = cfg.cd;
    this._level = this.level; // 缓存等级用于投射物参数
  }
}
```

**飞行逻辑伪代码**：

```
每帧更新每个projectile:
  if phase == 'out':
    1. 计算到目标的方向
    2. 将当前飞行方向朝目标方向旋转（最多trackAngle * dt）
    3. 沿弧线移动（加入curvature偏转）
    4. traveled += speed * dt
    5. 碰撞检测：检测路径上的敌人
       - 命中敌人（不在hitSet中）：造成伤害，加入hitSet，pierce-1
       - pierce <= 0：不穿透，跳过后续敌人
    6. 检查是否到达目标或maxDist：
       - 到达 -> phase = 'return'
       - traveled >= maxDist -> phase = 'return'

  if phase == 'return':
    1. 计算到玩家当前位置的方向
    2. 沿直线飞向玩家（returnSpeed）
    3. 碰撞检测：检测路径上的敌人（使用returnHitSet）
       - 命中敌人（不在returnHitSet中）：造成伤害，加入returnHitSet
    4. 到达玩家位置 -> 移除此projectile
```

**关键实现细节**：
- **追踪微调**：飞出阶段每帧最多偏转 `trackAngle * dt` 弧度，不是瞬间转向
- **弧线飞行**：在追踪方向基础上加入垂直分量偏移 `curvature`，创造弧线轨迹
- **双程命中独立计数**：飞出命中和飞回命中使用不同的hitSet，同一敌人可被飞出和飞回各命中一次
- **穿透独立**：穿透次数在飞出阶段消耗，飞回阶段有独立的穿透计数
- **多回旋镖目标分配**：Lv2投掷2个，分别追踪最近和次近的敌人；Lv3投掷3个，追踪最近/次近/第3近

## 8. 设计决策记录

1. **为什么选回旋镖作为第7种武器**：竞品调研P1第5项明确提出"新增2种基础武器（如回旋镖=回旋追踪、毒雾=持续伤害区）"。回旋镖的"追踪+双程"攻击模式是当前6种武器都不具备的，且视觉辨识度极高（V字形旋转飞行）。

2. **为什么双程伤害**：回旋镖在现实中的核心特征就是"飞出去再飞回来"，双程命中是最直觉的设计。这也让回旋镖成为唯一一个"单次攻击有两次命中机会"的武器，提供了独特的Build方向。

3. **为什么穿透从Lv2开始**：Lv1的回旋镖只有1个投射物，穿透0意味着只能命中1个敌人（飞出阶段）+ 1个敌人（飞回阶段）= 2次命中。Lv2/Lv3的穿透让回旋镖能穿过多个敌人，但需要更高的等级投资。

4. **为什么投掷间隔1.0s(Lv3)**：回旋镖的DPS已经是最高的（20），如果投掷间隔过短会过于强势。1.0s间隔意味着每次投掷约2秒完成（1s飞出+0.5s飞回+1s冷却），给玩家明确的节奏感。

5. **为什么追踪偏转角随等级增长**：Lv1的30度意味着需要大致瞄准方向，Lv3的60度意味着几乎全向追踪。这让回旋镖从"需要预判"成长为"自动追踪"，体感提升明显。

6. **为什么雷霆回旋用闪电链而非AOE**：闪电链保留了闪电武器的核心体验（随机电击），与回旋镖的追踪结合后形成"追踪→命中→连锁"的三段效果。AOE会与暴风雪/圣光领域重叠。

7. **为什么烈焰回旋用火焰轨迹而非范围DOT**：火焰轨迹创造了"移动的区域控制"效果 -- 回旋镖飞过的路径成为临时危险区域。这是全新的攻击维度，不与任何现有武器重叠。

8. **为什么给回旋镖设计磁铁协同而非护甲/生命结晶协同**：磁铁的"拾取范围扩大"与回旋镖的"飞行路径覆盖"有天然的互补关系。护甲/生命结晶与武器攻击模式无直接关联。暴击戒指则与回旋镖的"低频高伤"完美匹配 -- 每次命中的暴击收益最高。

9. **为什么没有回旋镖+幸运硬币协同**：幸运硬币（待实现）已有4个协同（命运赌徒/圣水炼金/炼金烈焰/冰霜拾荒），覆盖足够。为避免幸运硬币的协同密度过载，不在本次新增回旋镖+幸运硬币组合。未来可考虑"回旋镖+幸运硬币 -> 幸运转轮：击杀时有概率掉落额外宝箱"。

10. **为什么回旋镖的进化武器命名用英文后缀(-rang)**：保持与项目中其他进化武器命名风格一致（ThunderHolyWater/FireKnife等），同时-rang是回旋镖(boomerang)的自然后缀，读起来直观。

11. **为什么弧线飞行用curvature参数**：纯直线飞行缺乏"回旋镖"的视觉辨识度。curvature参数让飞行路径呈弧线，配合追踪微调创造出自然的回旋效果。Lv3的curvature从0.3降到0.2（更直的飞行），模拟"更熟练的投掷"。

12. **回旋镖对5分钟短局的平衡影响**：回旋镖在5分钟内最多升到Lv2（需要2次升级选择），DPS约17（4x2/1.4s），低于飞刀Lv3的15-20范围，不会打破现有平衡。在无尽模式中，回旋镖Lv3+进化会成为后期核心武器之一，鼓励玩家探索新的Build路径。

13. **为什么回旋镖的初始武器选择放到魔法师池中**：战士初始飞刀、游侠初始圣水，都不替换。回旋镖加入魔法师的"自选初始武器"池（从圣水/飞刀/闪电扩展为圣水/飞刀/闪电/**回旋镖**），增加了初始选择的多样性。
