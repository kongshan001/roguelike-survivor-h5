## 2026-04-03 — 策划迭代6：第二条进化路线 — 火焰飞刀

### 成果
**进化路线**：飞刀 Lv.3 + 火焰法杖 Lv.3 → 🔥🗡 火焰飞刀

| 属性 | 数值 |
|------|------|
| 投掷数量 | 5把（继承飞刀Lv3） |
| 飞行速度 | 280px/s（略快于普通飞刀250） |
| 命中伤害 | 3（与飞刀Lv3相同） |
| 穿透 | 2次（飞刀Lv3只有1次） |
| 燃烧DOT | 3/s持续2s（继承火焰法杖Lv3点燃） |
| 投掷间隔 | 0.5s（快于普通飞刀0.7s） |

**CONFIG**：
```javascript
{ a:'knife', b:'firestaff', result:'fireknife', name:'火焰飞刀', icon:'🔥🗡', desc:'燃烧穿透飞刀' }
```

**机制**：
- 继承飞刀的自动瞄准最近敌人+投掷
- 每把飞刀命中后附加燃烧效果（`enemy._burn={dmg:3,t:2}`）
- 燃烧叠加（多次命中重置持续时间）
- 穿透2次：穿过前2个敌人后消失
- 视觉：橙红色飞刀 + 命中爆火花

### 决策记录
- 火焰飞刀填补"高频多目标+持续伤害"的生态位
- 穿透+燃烧的组合让清怪效率显著提升，是进化武器的应有强度
- 与雷暴圣水（范围持续）形成互补

## 2026-04-03 — 策划迭代5：地图道具拾取 — 食物回血

### 成果
**食物回血系统设计**：

| 属性 | 数值 |
|------|------|
| 掉落概率 | 10%（每只敌人死亡） |
| 回复量 | 1 HP（1颗❤️） |
| 拾取范围 | 与宝石相同（35px快速吸入，全图漂移） |
| 存活时间 | 15秒后消失 |
| 上限 | 同屏最多8个食物 |
| Boss掉落 | 必掉3个食物 |

**食物类型**（纯视觉区分，效果相同）：
- 🍖 烤肉（僵尸掉落）
- 🍇 葡萄（蝙蝠掉落）
- 🧀 奶酪（骷髅掉落）

**CONFIG 数值**：
```javascript
FOOD: {
  dropRate: 0.1,       // 10% drop chance
  healAmount: 1,       // 1 HP
  maxFood: 8,          // max on screen
  lifetime: 15,        // seconds before despawn
  bossDropCount: 3     // boss guaranteed drops
}
```

**实现要点**：
- Food 类似 Gem 类，有位置/漂移/拾取逻辑
- 拾取时 `player.hp = min(player.hp + 1, player.maxHp)`
- 绘制：对应 emoji 或像素食物精灵
- 与 Gem 共用全图漂移机制

### 决策记录
- 食物回血是生存类游戏的基础机制，玩家HP上限8但无自然恢复，食物提供额外续航
- 10%掉落率意味着平均每10只怪掉1个食物，配合后期密集刷怪提供稳定回复
- Boss必掉3个食物，作为战斗中的"补给站"
- 回复量仅1HP，避免食物过强导致无敌
