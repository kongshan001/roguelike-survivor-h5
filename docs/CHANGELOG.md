# Changelog

All notable changes to 肉鸽幸存者 (Roguelike Survivor) will be documented in this file.

## [0.9.0] - 2026-04-03

### Added
- **音效系统**：Web Audio API 8-bit 合成音效，零外部文件
  - 11种音效：hit/kill/knife/lightning/levelup/pickup/chest/boss/freeze/gameover/victory
  - 4种渲染路径：白噪声(noise)、音阶序列(seq)、重复(repeat)、单音/滑音(default)
  - 音量控制 `masterVolume=0.3`
  - `AudioContext` 延迟创建，满足浏览器自动播放策略
- **12个音效触发点**覆盖全部核心游戏事件
- **音画联动规范**：每个音效对应明确视觉反馈

### QA
- E2E 测试 12/14 通过（2个时序相关 flaky，非回归）

## [0.8.0] - 2026-04-03

### Added
- **第6种武器：冰冻光环 (FrostAura)** — 范围减速+冰冻控制的控场型武器
  - Lv1: 80px范围, 30%减速, 1/s伤害
  - Lv2: 100px范围, 45%减速, 1.5/s伤害
  - Lv3: 130px范围, 60%减速, 2/s伤害 + 8%/秒概率冰冻1.5秒
  - 仅通过升级面板获取（不在初始选择池）
- **减速机制**：敌人 `_slow` 属性 + `_slowTimer` 刷新，移动速度×(1-slow)
- **冰冻机制**：Lv3解锁，冰冻中敌人完全停止行动
- **冰蓝色光环视觉**：脉动圆环 + 径向渐变 + 冰晶粒子旋转 + 脉冲波纹
- **减速/冰冻状态视觉**：减速=浅蓝覆盖，冰冻=深蓝覆盖+白色冰晶

### QA
- E2E 测试 12/14 通过（2个时序相关 flaky，非回归）

## [0.3.0] - 2026-04-03

### Added
- 受伤红屏闪烁效果 (`screenFlash`)
- 敌人死亡骷髅漂浮文字 (`dmgTexts`)
- 全图经验宝石磁吸系统 —— 所有宝石持续向玩家方向漂移，近距离加速吸入

### Fixed
- **[关键]** 修复经验宝石无法收集的问题 —— 宝石仅在被拾取前计算距离，移动后未重新检测导致永远无法到达玩家
- **[关键]** 修复DPR渲染偏移 —— `Camera.w2s()` 返回物理像素坐标，导致Retina屏幕上精灵位置错乱。改为返回逻辑像素坐标，统一所有绘制调用
- 修复 CSS `#hud` 中 `display:none` 与 `display:flex` 冲突
