# Changelog

All notable changes to 肉鸽幸存者 (Roguelike Survivor) will be documented in this file.

## [0.3.1] - 2026-04-03

### Fixed
- **[BUG-005]** 圣水伤害提升：碰撞判定 `8+e.w/2` → `12+e.w/2`，伤害因子 `dt*8` → `dt*15`，基础伤害 `1` → `1.5`
- **[BUG-006]** 磁铁被动重设计：从"拾取范围+30px"改为"经验获取量+30%（可叠加3次=+90%）"

### Changed
- 蝙蝠精灵从 10×10 放大至 14×14，增加翅膀细节和黄色眼睛（ENH-002）
- 圣经武器视觉重做：白色半透明矩形 → 金色光环 + 书页精灵，与圣水明显区分（ENH-001）

### QA
- E2E 测试 13/14 通过（1个时序相关 flaky）

## [0.3.0] - 2026-04-03

### Added
- 受伤红屏闪烁效果 (`screenFlash`)
- 敌人死亡骷髅漂浮文字 (`dmgTexts`)
- 全图经验宝石磁吸系统 —— 所有宝石持续向玩家方向漂移，近距离加速吸入
  - 拾取范围内：250px/s 快速飞行
  - 全图范围：40-100px/s 缓慢漂移（距离越近越快）

### Fixed
- **[关键]** 修复经验宝石无法收集的问题 —— 宝石仅在被拾取前计算距离，移动后未重新检测导致永远无法到达玩家
- **[关键]** 修复DPR渲染偏移 —— `Camera.w2s()` 返回物理像素坐标，导致Retina屏幕上精灵位置错乱。改为返回逻辑像素坐标，统一所有绘制调用
- 修复 CSS `#hud` 中 `display:none` 与 `display:flex` 冲突

## [0.2.0] - 2026-04-03

### Changed — 数值平衡调整
- 玩家 HP: 5 → 8
- 玩家移动速度: 150 → 160
- 无敌帧时长: 0.5s → 1.0s
- 拾取范围: 30px → 35px
- 宝石飞行速度: 200 → 250
- 飞刀冷却: 1.0s → 0.7s
- 敌人生成间隔全面放缓（早期 2.5s/只，后期 0.5s/3-4只）

### Fixed
- 地面渲染性能优化 —— 单次 `fillRect` 替代逐格绘制，减少 draw call

## [0.1.0] - 2026-04-02

### Added — 初始版本，完整可玩
- **Canvas 2D 像素风渲染**，单HTML文件，零外部依赖
- **5分钟限时生存** 模式（4:30 Boss出现）
- **4种武器系统**
  - 圣水 — 环绕旋转，可升级至3球
  - 飞刀 — 自动投掷，可升级至3把穿透
  - 闪电 — 随机电击，可升级至链式双击
  - 圣经 — 范围旋转，可升级至大范围双倍伤害
- **4种敌人**
  - 僵尸（慢速追踪）
  - 蝙蝠（快速冲锋）
  - 骷髅弓手（远程射击）
  - 骨龙Boss（双阶段，弹幕攻击）
- **升级系统** — 3选1面板（新武器/武器升级/被动道具）
- **3种被动道具** — 疾风靴/护甲/磁铁（每种最多叠加3次）
- **操控方式**
  - 移动端：虚拟摇杆触控
  - PC端：WASD/方向键
- **HUD界面** — 倒计时、等级、金币、血条、经验条、小地图
- **完整游戏流程** — 标题画面 → 武器选择 → 生存战斗 → 结算画面
- **时间递增难度曲线** — 每分钟提升敌人血量20%、速度10%
- **移动端适配** — DPR适配、禁止缩放、自适应分辨率
